纯内联样式演示：.none vs display: inline-block

场景 1: 正常显示

元素使用内联样式 style="display: inline-block; ..."。
它们会横向排列，且拥有宽高。

场景 2: 隐藏元素

中间的盒子加上了 data-pz="none" 。
即使它内联样式里写了宽高和颜色，它也会彻底消失，不占任何空间。

↑ 看到吗？Box 2 不见了，Box 1 和 Box 3 自动靠拢。

技术注记：
为了让 .none 类能覆盖内联样式中的 display: inline-block，CSS 定义中通常需要使用 !important。
已更新 CSS 为：.none { display: none !important; } (为了演示效果必须如此，否则内联样式优先级更高)。
